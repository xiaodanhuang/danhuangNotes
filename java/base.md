### Java基本概念
###### Java大法好
- 特性：面向对象，内存管理，跨平台可移植性顶呱呱。
- 工作方式：源代码->编译器->输出->JVM(java虚拟机)
- 程序结构：源文件=>类=>方法
- main()是程序运行的起点

###### 类与对象
- 类是对象的蓝图，对象是类的实例。
- 对象包括实例变量以及方法。

###### primitive主数据类型和引用
- 存放数值： byte short int long  float double
- 字符：char
- 布尔 boolean
- 对象引用变量保存的是存取对象的方法，类似指向对象的指针。

###### 对象的行为
- java是值传递。
- java的get是返回实例变量的值，set是设置实例变量的值
